# Amali Detalles — Tienda Online de Detalles Personalizados

**Live Demo** → https://amali-detalles.vercel.app  
**Panel Admin (protegido)** → https://amali-detalles.vercel.app/admin

Proyecto real 100% funcional entregado a cliente (Quito, Ecuador).  
E-commerce completo con carrito, subida de productos, filtros por categoría y panel de administración seguro.

## Tecnologías utilizadas

### Frontend
- React 18 + Vite
- Tailwind CSS
- React Router
- Context API (Auth + Carrito)
- Cloudinary (imágenes)

### Backend
- Node.js + Express
- MongoDB Atlas
- JWT Authentication
- Cloudinary Storage (multer-storage-cloudinary)
- Deploy: Render.com

## Características principales

- Catálogo con filtros por categoría (Fresas, Ramos, Cumpleaños, etc.)
- Carrito de compras persistente + botón directo a WhatsApp
- Panel de administración protegido con login (solo la dueña)
- Subida/edición/eliminación de productos con imágenes
- Diseño 100% responsive y romántico
- Deploy gratuito y escalable (Vercel + Render)

## Deploy

- Frontend → Vercel: https://amali-detalles.vercel.app
- Backend → Render: https://amali-detalles-api.onrender.com
